# CirclePlatformer
<body>
Unity project from 2021. A platformer game with progression in the form of unlocking new abilities.

<h3> Instructions </h3>
<p>
Can be played from Unity Play (Unity's own hosting platform): <br>
https://play.unity.com/en/games/224df4fe-77c9-4a19-aab8-697431e47f28/circle-platformer
<h4>Controls:</h4>
<b>A/D</b> - move left / right <br>
<b>W</b> - jump <br>
<b>E</b> - hover  <br>
<b>Space bar</b> - wall phase  <br>
</p>

<h2>DESCRIPTION</h2>
<p>
<b> Platform through 30 levels, </b>unlocking abilities every 10 levels. Use all your abilities to figure out how to complete each level. <br>



</p>
