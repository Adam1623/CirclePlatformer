# CirclePlatformer
<body>
Unity project from 2021. A platformer game with progression in the form of unlocking new abilities.

<h3> Instructions </h3>
<p>Run "Main.py" to run the game.

<h4>Controls:</h4>
<b>a/d</b> - move left / right <br>
<b>w</b> - jump <br>
<b>Left click release</b> - lightning strike <br>
</p>
